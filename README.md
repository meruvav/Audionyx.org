# Audionyx

A simple React + Vite audio utility site using Ubuntu font and plain hue-based styling.

## Included traffic pages

- `/hearing-test`
- `/tone-generator`
- `/sleep-frequencies`
- `/frequency-guide`
- `/support`

## Included tool dashboard blocks

- Multiple Tone Generator
- Pitch Shifter
- Time Stretcher
- Voice Generator
- Voice Recorder
- Sweep Generator
- Instrument Tuning
- Subwoofer Testing
- Hearing Test
- Alarm Generator
- Noise Generator
- Binaural Beats
- 432Hz preset
- DTMF Signals
- Metronome
- The Pips

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Important notes

- The PayPal donation form uses a placeholder email. Replace it before going live.
- Browser pitch shifting and stretching are limited. This is not studio-grade DSP.
- Affiliate links are placeholders and need real partner links before launch.
